## ✨ Features

- 🏠 **Restaurant Browsing** — Explore a list of available restaurants and cuisines
- 📋 **Menu Viewing** — Browse detailed menus with item names, descriptions, and prices
- 🛒 **Cart Management** — Add, remove, and update item quantities in real time
- 📦 **Order Placement** — Submit orders with a smooth checkout flow
- 📱 **Responsive UI** — Fully optimized for desktop, tablet, and mobile screens
- ⚛️ **Component-Based Architecture** — Modular React components with clean state management

## ⚙️ Getting Started

### Prerequisites

- Node.js v16+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/rohanshan2006/vingo.git

# Navigate into the project
cd vingo

# Install dependencies
npm install

# Start the development server
npm start
```

The app will run at `http://localhost:3000`
